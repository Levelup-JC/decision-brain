### 截图生成方式

使用 Playwright 1.61.1 自动化截图，脚本: `take-screenshots.mjs`。

- 桌面视图: 1440x1000 viewport，展示三栏布局
- 手机视图: 390x844 viewport，验证移动端适配
- 资产操作: 通过 `/api/manage-position` 写入 SOL 仓位后截图
- 发光曲线: 在对话中输入"SOL 趋势"触发图表渲染
- 持仓总览: 在对话中输入"我的持仓总览"触发资产面板

所有截图均在本地服务器 `http://127.0.0.1:4177` 运行状态下生成。
